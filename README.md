SBA-307 project
This project name is K-mania.KMania is an interactive platform designed to make preparation for web development interviews simple and fun.
We focus on providing HTML and CSS Interview Questions for beginners and enthusiasts.
# Architecture Overview
Frontend

Built with HTML + CSS
**#Menu**
**Home** Home page
**About Us** -small description of who we are
In **Topics** dropdown menu you have HTML and CSS options.
   If HTML is selected, you will be redirected to HTML page where you can access HTML related questions.
   If CSS is selected, you will be redirected to HTML page where you can access CSS related questions.
**Contact Us** -Form to contact us and subscribe for future updates.
